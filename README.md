# EnhancedVanilla

Bukkit plugin to add several enhancements to Minecraft in the spirit of the base Vanilla gameplay.

All mechanics are individually configurable in the config.

## Included Mechanics

- [x] TreeCutter
- [x] ChickenShearing
- [x] LadderDropDown
- [x] MinecartBlockPlacement
- [x] ShaveSnowLayers
- [x] BetterSilkTouch
- [x] DoorKnocking
